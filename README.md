# Music Cleaner

A utility to clean up music file tags.

Supported file types:

- MP3

Tag clean up:

- Album artists

## Usage

```shell
MusicCleaner [DIR]
```

### Argument

- **DIR**: The music directory to process.

### Options

- `-v`, `--version`
  - Display the application version.
- `-h`, `--help`
  - Display the help.

### Directory Structure

```text
- Artist 1
  - (year) Album 1
  - (year) Album 2
  - ...
- Artist 2
  - (year) Album 1
  - ...
- ...
```
